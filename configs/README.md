# Diretório de Configurações

## Arquivos

- **`config.example.yaml`**: Modelo de configuração (não editar)
- **`config.yaml`**: Configuração ativa (editar conforme necessário)

## Como usar

1. Copie `config.example.yaml` para `config.yaml` (se ainda não existir)
2. Edite `config.yaml` para ajustar:
   - Caminho do arquivo de entrada
   - Colunas numéricas e categóricas
   - Parâmetros de pré-processamento
   - Métodos de clustering a executar
   - Parâmetros de visualização

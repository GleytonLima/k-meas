# Código Fonte

## Módulos

- **`run_pipeline.py`**: Entrypoint principal que orquestra todo o pipeline
- **`preprocess.py`**: Pré-processamento de dados (missing, scaling, encoding)
- **`clustering.py`**: Implementação dos métodos de clustering (k-means, GMM, hierárquico)
- **`visualization.py`**: Geração de gráficos e visualizações
- **`reporting.py`**: Geração de relatórios e insights

## Execução

O pipeline é executado via Docker (ver `docker-compose.yml`).

# Diretório de Dados

Coloque aqui o arquivo CSV de entrada.

## Arquivo esperado

- **Nome:** `input.csv`
- **Formato:** CSV com cabeçalho
- **Encoding:** UTF-8
- **Separador:** Vírgula (configurável no config.yaml)
- **Decimal:** Ponto (configurável no config.yaml)

## Estrutura mínima recomendada

- Coluna de identificação (ex: `id_contrato`)
- Variáveis numéricas (ex: `valor_emprestimo`, `taxa_juros_anual`)
- Variáveis categóricas (ex: `nome_produto`)

Consulte `configs/config.yaml` para configurar quais colunas usar.
